# ⚛️ Физика Курсы - 7-9 Сынып

## Құрылымы

Бұл - **веб-негізді толық Физика платформасы**, орта мектептің 7-9 сынып студенттеріне арналған.

### 📚 Модульдер

1. **🔵 Механика** - Қозғалыс, күш, энергия
2. **🟢 Кинематика** - Қозғалыстың математикалық сипаттамасы
3. **🟡 Молекулалық физика** - Молекулалар және газдар теориясы
4. **🔴 Термодинамика** - Жылу және температура
5. **🟣 Электр және магнетизм** - Электр заряды және магнит өрістер
6. **🟠 Оптика** - Жарық және линзалар
7. **⚫ Квантық физика** - Фотондар және атомдар

### ✨ Ерекшеліктері

✅ **Қатысушылар кіру/тіркеу** - Өздіңіз есептік жазба құра аласыз
✅ **Прогрес сақтау** - Өндіктен өтті сабақтарын есіңізде сақтайды
✅ **Өтті/өтпеді статусы** - Әр сабақ үшін өтті деп белгілей аласыз
✅ **Формулалар** - Әр сабақта физикалық формулалар бар
✅ **Орташа деңгейдегі тестілер** - Өндіктен білім тексеріңіз
✅ **Қатысушылар статистикасы** - Барлық пайдаланушыларды есеп

## 🚀 Қалай пайдалану керек

1. `index.html` файлын браузерде ашыңыз
2. Есептік жазба құрыңыз (кіру)
3. Сол жақтан модульді таңдаңыз
4. Сабақты оқыңыз
5. Формулаларды өндіктіңіз
6. Тестілерді өтіңіз
7. "Өтті деп белгілеу" батырмасын басыңыз
8. Прогресіңізді қараңыз

## 📝 Файлдар

- **index.html** - Негізгі HTML файлы
- **styles.css** - Стильдер және дизайн
- **app.js** - Интерфейс логикасы
- **auth.js** - Авторизация және есептік жазба
- **data.js** - Барлық физика курсының деректері

## 🔧 Техникалық мәліметтер

- **Қолданатын технологиялар:** HTML5, CSS3, JavaScript (Vanilla)
- **Сақтау:** LocalStorage (браузер ішінде)
- **Тіл:** Қазақ тілі

## 📊 Статистика

- **Сабақтар:** 20+ шт
- **Формулалар:** 50+ шт
